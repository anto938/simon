# simon
Game for the STM32F303 Microcontroller, written in ARM Assembly (Cortex-M4) and based off of Hasbro's "Simon". Was done as an assignment for the module "Introduction to Computing II" from the first year of the Computer Science B.A. at Trinity College, Dublin.
